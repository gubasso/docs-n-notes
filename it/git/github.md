- GitHub fork naming pattern

```
<organization>-<project-name>-fork
```

Examples:

```
suse-openscap
myco-website
acme-inventory-fork
```

```
SUSE-Enceladus-keg-fork
forked from SUSE-Enceladus/keg
```
