# Email Server

Ports 25, 465, 587

- 25 (SMTP)
- 587 (MSA/STARTTLS)
- 465 (MSA/TLS)
- 993 (IMAPS)
- 4190 (WebSieve)

Check if port 25 is open:

```
telnet portquiz.net 25
```

If this connects to port 25, the outbound is opened at port 25.
