# docs-n-notes

Personal notes and documentations for a bunch of unrelated topics.

## Structure

`personal/` holds life / non-tech notes. Each top-level area reserves a domain — put new content
where its domain matches (the directories themselves are the source of truth for what currently
exists):

- **`personal/`** — life / non-tech (health, notes from articles and videos).

The tech knowledge base has been migrated out of this repository to `exobrain-tech` (public) and
`exobrain-tech-vault` (private).
