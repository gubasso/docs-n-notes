# riptask (`tsk`)

Notes for the `riptask` plaintext issue tracker (`tsk` binary).

- [`commands.md`](commands.md) — tsk CLI specification: preflight checks, top-level command
  overview, and the full `tsk new` reference (flags, long-body handling, output format). Shared
  source of truth for the `tsk-new`, `tsk-impl`, and `prex` Claude skills.

The `tsk` binary is authoritative; these notes capture stable usage patterns. When in doubt, run
`tsk <command> --help`.
