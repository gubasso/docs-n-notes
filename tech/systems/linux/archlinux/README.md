# Arch Linux

Arch Linux install and setup notes — installation (including reinstall on existing LUKS/LVM), boot
loader, post-install, the US AltGr-International TTY keymap, and SSH agent live as topic files here;
the [`AGENTS.md`](./AGENTS.md) Source Map indexes them file by file.

Machine-specific package manifests remain in dotfiles under `_docs/os/archlinux/packages/`.
